# Тестовое задание

**Подготовка окружения**  
- **заметка** - разработка была на дистрибутиве GNU/Linux, инструкция по установке окружения и т.д может незначительно отличаться. Если возникнут трудности - необходимо изучить методы установки для своей ОС самостоятельно  
Клонируем репозиторий
```sh
git clone https://github.com/rickert156/diamant_task && cd diamant_task
```
Поставим окружение и пакеты
```sh
python3 -m venv venv && ./venv/bin/pip install -r package.txt
```
Активируем окружение
```sh
source venv/bin/activate
```

## Проверка email-доменов
Для проверки доменов используем скрипт check_mx.py. Условия использования скрипта:  
- аргументом указываем файл csv с имейлами
- в csv файле должна быть колонка Email  
Пример использования:
```sh
python3 check_mx.py email.csv
```
